Welcome to Kata Collection 2!

This assessment was created with number sense and code
knowledge in mind, to provide a learning and practice experience.

The criteria for these Katas are as follows:

  1. Display the numbers from 1 - 20
  2. Display the even numbers from 1 to 20
  3. Display the odd numbers from 1 to 20
  4. Display the multiples of 5, up to 100
  5. Display the square numbers of 1 to 10 (1, 4, ..., 100)
  6. Display the numbers from 20 - 1, counting down
  7. Display the even numbers from 20 - 1, counting down
  8. Display the odd numbers from 20 - 1, counting down
  9. Display the multiples of 5, counting down from 100
  10. Display the square numbers, counting down from 100
  11. Display all elements of the sample array
  12. Display only the even numbers of the array
  13. Display only the odd numbers of the array
  14. Display the square of each number in the array
  15. Display the sum of all numbers from 1 - 20 (e.g. 1 + 2 + ... + 20)
  16. Display the sum of all numbers in the array
  17. Display the smallest number in the sample array
  18. Display the largest number in the array
  19. Display 20 solid gray rectangles, 20px high and 100px wide
  20. Display 20 solid gray rectangles, 20px high, with widths starting at 105px and increasing by 5px to a max of 200px
  21. Display 20 solid gray rectangles, 20px high, with widths in pixels given by the values in the sample array
  22. As with #21, but with every other rectangle being solid red
  23. As with #21, but with rectangles with even widths being red
